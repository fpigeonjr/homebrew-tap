# homebrew-tap

Homebrew tap for [fpigeonjr](https://github.com/fpigeonjr) tools.

## Install

```bash
brew tap fpigeonjr/homebrew-tap
brew install mfp
```

## Formulae

| Formula | Description |
|---------|-------------|
| `mfp` | Terminal UI for [musicforprogramming.net](https://musicforprogramming.net) |

## Usage

```bash
mfp          # launch
mfp --version  # check version
```

> **Prerequisite:** `brew install mpv` (installed automatically as a dependency)

## Updating

```bash
brew update
brew upgrade mfp
```

## Uninstall

```bash
brew uninstall mfp
brew untap fpigeonjr/homebrew-tap
```
